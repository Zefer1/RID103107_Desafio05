## 💻 Como executar localmente

### 1) Backend (API)

```bash
cd desafio05-api
npm install
npm run dev   # nodemon index.js → API em http://localhost:3001

2) Front-end (React)

cd Desafio05-Front
npm install
npm run dev   # Vite → front em http://localhost:5173

🧰 Tecnologias

Back-end: Node.js, Express.js, CORS

Front-end: React, Vite, React Router, Axios
